# 📊 Dmart Sales Analysis Dashboard

## 📌 Project Overview

The **Dmart Sales Analysis Dashboard** is an Excel-based data analytics project designed to analyze sales performance, customer trends, and product insights using interactive dashboards.
This project demonstrates how raw sales data can be transformed into meaningful business insights using **Microsoft Excel**, formulas, Pivot Tables, and data visualization techniques.

The dashboard helps in:

* Tracking sales performance
* Identifying top-performing products
* Understanding customer behavior
* Monitoring regional and category-wise sales trends
* Supporting data-driven decision-making

---
